model_fit module¶

Contains methods to fit gene expression to mixed gaussian model using the Expectation Maximization algorithm

model_fit.fit_test_model(x)[source]¶

Given a series of samples, representing gene expression, returns a 2-component gaussian mixture model fitted to the samples. Expectation-Maximization is used to fit the data

Parameters:x (list) – a list of floats representing gene expression profile samples for one gene. each entry in the list should be vectorized! ie. [[1.0], [3.2], [4.3]]
Returns:a sklearn.mixture.GMM, which represents a two component gaussian mixture model
model_fit.get_trained_models(gene_profiles)[source]¶

Given a mapping of genes to expression values, returns a gaussian model for each gene

Parameters:gene_profiles (dict) – a dictionary of strings representing gene names to lists of floats representing expression values
Returns:a dict mapping gene names to the trained model of the gene
model_fit.print_model_params(gauss_model)[source]¶

Prints out the coeff, mean, and standard deviation of the given mixture model. Also prints out information related to how the model was built

Parameters:model (sklearn.mixture.GMM) – the mixture model of a given gene
