master_selection module¶

This file contains a series of scripts for evaluating gene models and determining their usability as a “master gene”

master_selection.calculate_bayes_error(model)[source]¶

Returns the bayes error of the given mixture model. This is calculated by integrating the false class density around the decision boundary of the model.

Requires:the model is composed of two gaussian components!
Parameters:model (sklearn.mixture.GMM) – the mixture model of a given gene
Returns:the bayes error of the classifier as a float

master_selection.findDecisionBoundary(mu1, sigma1, mu2, sigma2)[source]¶

Given the parameters describing two gaussian models representing two classes, returns the decision boundary for classification. THis assumes classification is done through a z-score.

Requires:

the parameters given describe two different standard deviations
Parameters:
  • mu1 (float) – the mean of the first distribution
  • sigma1 (float) – the standard deviation of the first distribution
  • mu2 (float) – the mean of the second distribution
  • sigma2 (float) – the standard deviation of the second distribution

:returns the decision boundary as a float

master_selection.findIntersection(mu1, sigma1, mu2, sigma2)[source]¶

Finds and returns the intersection points of two standard deviations

Requires:

the parameters given describe two different standard deviations
Parameters:
  • mu1 (float) – the mean of the first distribution
  • sigma1 (float) – the standard deviation of the first distribution
  • mu2 (float) – the mean of the second distribution
  • sigma2 (float) – the standard deviation of the second distribution
Returns:

the intersection points of the two distributions as a tuple pair. If there is only one intersection, the point appears twice in the tuple.
master_selection.solveQuadratic(a, b, c)[source]¶

Given coefficients a, b, and c. Solves the quadratic in the form \(ax^2 + bx + c = 0\).

Requires:

the determinant of the quadratic is non-zero
Parameters:
  • a (float) – the coefficient of the x^2 term
  • b (float) – the coefficient of the x term
  • c (float) – the coefficient of 1 term
Returns:

a tuple with two values containing two roots of the equation. If there is only one distinct root, it will appear twice
